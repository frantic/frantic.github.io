# Project Notes

This is Alex Kotliarskyi's blog (frantic.im).

## Build System

- Node.js-based static site generator (custom Jekyll replacement)
- Build: `touch .env && ./scripts/build` from project root
- Dev server: `node scripts/make.js --dev` (serves on port 9099)
- Output goes to `_build/`
- Posts live in `_posts/` and `_notes/`
- Layouts in `_layouts/`, includes in `_includes/`, pages in `pages/`
